This repository includes the materials for the 'rover' application EBPDs. 

1) the concrete planning operators ('ebpd\domain.ebpd')
2) the abstract planning operators ('ebpd\domain-abs.ebpd')
3) the abstraction hierarchies ('ebpd\abstraction-hierarchies.ebpd')
4) a set of problems ('ebpd\p#.ebpd')
5) the problem sets obtained by the tvla (scope inference) ('probsets/').
   each probset contains the problems; 2-valued and 3-valued structures 
   built by the TVLA; an experience for the problem set; and an activity
   schema for solving all problems in the set.
