This repository includes the supplementary materials for the proposed 
experience-based learning and planning system in our experiments. It includes 
the 'stacking-blocks' and 'cafe' application EBPDs. Each EBPD contains:

1) the planning operators domain ('ebpd\domain.ebpd')
2) the abstract operators domain ('ebpd\domain-abs.ebpd')
3) the abstraction hierarchies ('ebpd\abstraction-hierarchies.ebpd')
4) a set of experiences ('experiences\experience#.ebpd')
5) a set of activity schemata ('schemata\schema#.ebpd')
6) a set of problems ('ebpd\p#.ebpd')
